# 支出管理アプリ（ベージュデザイン版）

温かみのあるベージュ系配色の支出管理Webアプリケーションです。

## 特徴

### デザイン
- **ベージュ系カラーパレット**: 落ち着いた温かみのある配色
- **やわらかいフォント**: ヒラギノ角ゴ、Noto Sans JP等の視認性の高い日本語フォント
- **大きな入力フィールド**: 入力しやすい十分なサイズ感
- **レスポンシブ対応**: スマートフォンでも快適に操作可能

### 機能
- 日付・カテゴリ・金額・メモの入力
- カテゴリ別の支出集計
- 全支出の合計表示
- CSV形式でのデータエクスポート
- LocalStorageでのデータ永続化

## 使い方

### 1. ファイルのダウンロード
以下の3ファイルをダウンロードして、同じフォルダに配置してください：
- `index.html`
- `style.css`
- `main.js`

### 2. 起動
`index.html` をブラウザで開くだけで使用できます。

### 3. 基本操作
1. **支出の追加**: フォームに日付、カテゴリ、金額、メモを入力して「追加」ボタンをクリック
2. **支出の確認**: 下部の「支出履歴」セクションで一覧表示
3. **支出の削除**: 各項目の「✕」ボタンをクリック
4. **データのエクスポート**: 「CSVコピー」ボタンでクリップボードにコピー
